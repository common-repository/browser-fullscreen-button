Add the shortcode [bfb] on any pages or posts. And you will get 2 Buttons one opens your website in browser fullscreen and the other is for esc.
== Description ==

This Plugin is a very simple Plugin and allows you to add with the shortcode [bfb] buttons to your website to allow user to turn your website in fullscreen mode.

== Frequently Asked Questions ==

=How can i use it?=

Just activate the plugin and put the shortcode [bfb] where you want the button to be displayed.
